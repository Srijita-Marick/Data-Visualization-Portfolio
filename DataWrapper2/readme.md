# Data Theme: Higher Education 

## About
The data set for this directory is a combined data set that includes data from the 
[Government of Canada](https://open.canada.ca/data/en/dataset/90115b00-f9b8-49e8-afa3-b4cff8facaee/resource/b663a97c-61d4-4e17-af51-0cba01ef3a44) 
and the [World Bank](https://datacatalog.worldbank.org/search/dataset/0037712/world-development-indicators). The 
Government of Canada data provided the yearly totals of Country of Citizenship for study permit holders in Canada and 
the World Bank data provides the country region as well as their economic classification for the county in 2025. It also 
provides the region the country is part of according to the World Bank.
